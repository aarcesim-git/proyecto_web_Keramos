INSTRUCCIONES PARA ABRIR EL PROYECTO

Este proyecto usa fetch() para cargar noticias desde un archivo JSON externo.

Para que las noticias se carguen correctamente, es necesario abrir el proyecto con un servidor local, como Live Server en VS Code o cualquier otro servidor local.

Si abres el archivo index.html directamente haciendo doble clic (ruta file://), las noticias no se cargarán debido a las políticas de seguridad del navegador.

Gracias por la comprensión.
